# Gemini Grounding Remote MCP Server

Gemini APIのGrounding機能を使用したMCP（Model Context Protocol）サーバーです。

## 概要

このプロジェクトは、Google Gemini APIのGrounding機能を活用して、リモートでMCPサーバーを提供するものです。

## 必要な環境

- Node.js
- Google Gemini API キー

## インストール

```bash
npm install
```

## 使用方法

```bash
npm start
```
